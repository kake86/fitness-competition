## Fastest Way to Get Your App Online (5 minutes)

### Step 1: Install Vercel CLI
```powershell
npm install -g vercel
```

### Step 2: Build Your App
```powershell
cd C:\Users\kangl\planning-tool
npm run build
```

### Step 3: Deploy
```powershell
vercel
```
- Press Enter for all prompts (defaults are fine)
- When asked "Set up and deploy?", type `Y`
- When asked "Link to existing project?", type `N` (first time only)

### Step 4: Get Your URL
Vercel will show you a URL like: `https://your-app-name.vercel.app`
