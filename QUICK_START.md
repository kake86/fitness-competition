# Quick Start Guide

Get the Planning Tool running in 3 steps (if you already have Node.js installed).

## Prerequisites Check

First, verify you have Node.js installed:

```powershell
node --version
npm --version
```

## 3 Steps to Run

### 1. Navigate to Project
```powershell
cd C:\Users\kangl\planning-tool
```

### 2. Install Dependencies
```powershell
npm install
```
*(Wait 1-3 minutes for packages to download)*

### 3. Start the App
```powershell
npm run dev
```

## Open in Browser

Copy the URL shown in terminal (usually `http://localhost:5173`) and open it in your browser.
